# Q-Macro 政策翻译官开发方案 — LLM 解读政策，驱动主题轮动

## 目标

构建 `policy_interpreter.py`，从 `govcn_2025.csv` 提取政策信号，输出可执行的 ETF 主题推荐。

## 原则

强约束、可解释、带 fallback 机制。

## 输出

直接对接 `portfolio_builder.py` 的 `current_theme` 参数。

---

## 一、文件路径

- **智能体代码**：`Q-Macro/src/agents/policy_interpreter.py`
- **输入数据**：`Q-Macro/data/policy_texts/govcn_2025.csv`
- **依赖模块**：`src/core/macro_regime.py`（提供宏观上下文）

## 二、核心功能设计

### 输入

| 参数 | 类型 | 说明 |
| :--- | :--- | :--- |
| `policy_path` | str | 政策 CSV 路径（如 `data/policy_texts/govcn_2025.csv`） |
| `target_date` | str | 目标日期（如 `"2025-12-31"`） |
| `macro_context` | dict | 来自 `macro_regime.py` 的宏观状态 |

### 输出

```python
{
  "use_thematic": True,
  "recommended_theme": "高端制造",  # 必须是预定义子主题之一
  "confidence": 0.85,             # LLM 自评置信度（0~1）
  "evidence": ["中央经济工作会议强调新质生产力", "..."]  # 支撑依据
}
```

### 约束条件

- **主题必须来自 LLM 标签体系中的股票子主题**（见下表）
- **若无明确信号 → `use_thematic=False`**
- **失败时自动 fallback 到硬编码规则**

## 四、Prompt 设计（精准 + 防幻觉）

```text
你是一名资深宏观策略分析师，请基于以下政策文本和宏观背景，严格按要求输出JSON。

【当前宏观状态】
- 宏观阶段: {regime_label}
- 权益友好度评分: {equity_score}
- 增长动量: {growth_momentum}
- 通胀动量: {inflation_momentum}

【政策文本摘要】（仅包含2025年12月相关新闻）
{text_snippets}

【分析任务】
1. 判断当前政策是否明确支持某一投资主题
2. 若支持，从以下列表中选择最匹配的主题：
   {allowed_themes_str}
3. 若不支持任何主题，返回 use_thematic=false

【输出规则】
- 只输出一个JSON对象
- 字段:
  - use_thematic: boolean
  - recommended_theme: string（仅当use_thematic=true时有效）
  - confidence: float（0.0~1.0，基于证据强度）
  - evidence: array of strings（直接引用原文关键句，最多3条）
- 禁止编造不在列表中的主题
- 禁止输出任何其他文字

【示例输出】
{"use_thematic": true, "recommended_theme": "高端制造", "confidence": 0.9, "evidence": ["中国以系统化的顶层设计推动科技创新", "中国在制造业和技术创新领域处于领先地位"]}
```

## 五、代码结构（`policy_interpreter.py`）

```python
# src/agents/policy_interpreter.py

from pydantic import BaseModel, Field, validator
from typing import List, Optional
import pandas as pd
from langchain_openai import ChatOpenAI
from langchain_core.output_parsers import PydanticOutputParser
from langchain_core.prompts import ChatPromptTemplate
from dotenv import load_dotenv
import os

# 加载环境变量
load_dotenv()

# 合法主题列表（与ETF标签对齐）
ALLOWED_THEMES = [
    "人工智能", "半导体", "新能源车", "光伏", "医药", "消费",
    "金融", "周期", "高端制造", "数字经济", "科技创新", "新材料",
    "生物医药", "通用航空", "国企改革", "红利低波", "农业",
    "公用事业", "电力", "低碳经济", "碳中和", "物联网", "教育",
    "养老产业", "ESG", "可持续发展", "传媒", "房地产", "物流"
]

class PolicySignal(BaseModel):
    use_thematic: bool
    recommended_theme: Optional[str] = None
    confidence: float = Field(ge=0.0, le=1.0)
    evidence: List[str]
    
    @validator('recommended_theme', always=True)
    def validate_theme(cls, v, values):
        if values.get('use_thematic') and v not in ALLOWED_THEMES:
            raise ValueError(f"Theme '{v}' not in allowed list")
        return v

def load_recent_policy_snippets(policy_path: str, target_date: str, window_days: int = 30) -> str:
    """加载目标日期前N天的政策文本"""
    df = pd.read_csv(policy_path)
    df['date'] = pd.to_datetime(df['date'])
    target_dt = pd.to_datetime(target_date)
    start_dt = target_dt - pd.Timedelta(days=window_days)
    
    recent = df[(df['date'] >= start_dt) & (df['date'] <= target_dt)]
    
    # 拼接 content 字段（去重）
    snippets = []
    seen = set()
    for _, row in recent.iterrows():
        content = str(row['content']).strip()
        if content and content not in seen:
            snippets.append(content[:500])  # 截断防超长
            seen.add(content)
    
    return "\n".join(snippets[-5:])  # 最多5篇

def interpret_policy(
    policy_path: str,
    target_date: str,
    macro_context: dict,
    max_retries: int = 2
) -> dict:
    """主函数：调用LLM解析政策信号"""
    try:
        # 1. 加载近期政策片段
        text_snippets = load_recent_policy_snippets(policy_path, target_date)
        if not text_snippets:
            print("No recent policy text found, using fallback")
            return _fallback_signal(macro_context)
        
        # 2. 初始化LLM
        llm = ChatOpenAI(
            model=os.getenv("MODEL_NAME", "qwen-max"),
            temperature=0.0,
            api_key=os.getenv("LLM_API_KEY", "EMPTY"),
            base_url=os.getenv("LLM_API_BASE")
        )
        
        # 3. 构建Prompt
        parser = PydanticOutputParser(pydantic_object=PolicySignal)
        prompt = ChatPromptTemplate.from_messages([
            ("system", "你是一名资深宏观策略分析师，请基于政策文本和宏观背景，严格按要求输出JSON。"),
            ("user", """
【当前宏观状态】
- 宏观阶段: {regime_label}
- 权益友好度评分: {equity_score}
- 增长动量: {growth_momentum}
- 通胀动量: {inflation_momentum}

【政策文本摘要】
{text_snippets}

【分析任务】
1. 判断当前政策是否明确支持某一投资主题
2. 若支持，从以下列表中选择最匹配的主题：
   {allowed_themes_str}
3. 若不支持任何主题，返回 use_thematic=false

【输出规则】
- 只输出一个JSON对象
- 字段:
  - use_thematic: boolean
  - recommended_theme: string（仅当use_thematic=true时有效）
  - confidence: float（0.0~1.0，基于证据强度）
  - evidence: array of strings（直接引用原文关键句，最多3条）
- 禁止编造不在列表中的主题
- 禁止输出任何其他文字

【示例输出】
{"use_thematic": true, "recommended_theme": "高端制造", "confidence": 0.9, "evidence": ["中国以系统化的顶层设计推动科技创新", "中国在制造业和技术创新领域处于领先地位"]}
""")
        ])
        
        full_prompt = prompt.partial(
            format_instructions=parser.get_format_instructions(),
            allowed_themes_str=", ".join(ALLOWED_THEMES)
        )
        
        # 4. 调用链
        chain = full_prompt | llm | parser
        result = chain.invoke({
            "regime_label": macro_context.get("regime", "unknown"),
            "equity_score": macro_context.get("equity_friendly_score", 0.5),
            "growth_momentum": macro_context.get("growth_momentum", 0.0),
            "inflation_momentum": macro_context.get("inflation_momentum", 0.0),
            "text_snippets": text_snippets
        })
        
        return result.dict()
        
    except Exception as e:
        print(f"Policy interpreter failed: {e}, using fallback")
        return _fallback_signal(macro_context)

def _fallback_signal(macro_context: dict) -> dict:
    """硬编码备用方案（原signal_fusion逻辑）"""
    regime = macro_context.get("regime")
    score = macro_context.get("equity_friendly_score", 0)
    
    if regime == "recovery" and score > 0.6:
        return {"use_thematic": True, "recommended_theme": "人工智能", "confidence": 0.7, "evidence": ["Fallback rule"]}
    elif regime == "overheat":
        return {"use_thematic": True, "recommended_theme": "周期", "confidence": 0.7, "evidence": ["Fallback rule"]}
    else:
        return {"use_thematic": False, "recommended_theme": None, "confidence": 0.5, "evidence": ["Fallback: no thematic signal"]}
```
