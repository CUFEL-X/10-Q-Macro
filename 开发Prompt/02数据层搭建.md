# 宏观数据获取与存储方案

## 方案目标

本方案旨在通过 AKShare 库获取中国宏观经济关键指标数据，并建立标准化的数据存储结构，为 Q-Macro 策略系统提供可靠的基础数据支持。

---

## 数据获取范围

计划获取以下核心宏观经济指标数据：

1. **PMI（采购经理人指数）**：反映制造业和非制造业活动水平
2. **CPI（消费者价格指数）和 PPI（生产者价格指数）**：衡量通货膨胀水平
3. **社融（社会融资规模）**：反映金融体系对实体经济的支持程度
4. **M0/M1/M2（货币供应量）**：衡量货币市场流动性
5. **工业增加值**：反映工业生产活动水平
6. **GDP（季度）**：反映整体经济增长状况
7. **MLF（中期借贷便利操作）**：反映货币政策取向
8. **LPR（贷款市场报价利率）**：反映市场资金成本
9. **存款准备金率**：反映货币政策松紧程度

## 实现方案

### 脚本设计

创建 `scripts/fetch_macro_data.py` 脚本，实现数据的自动获取与存储：

#### 1. 导入必要库

```python
import akshare as ak
import pandas as pd
from datetime import datetime
import os
```

#### 2. 定义数据获取函数

```python
def fetch_and_save_data():
    """
    从AKShare获取宏观经济数据并保存到本地文件
    """
    # 确保数据存储目录存在
    os.makedirs("data/macro_data", exist_ok=True)
    
    # 记录执行时间
    current_datetime = datetime.now()
    print(f"开始获取宏观数据: {current_datetime}")
```

#### 3. 获取并存储 PMI 数据

```python
    # 获取PMI数据
    pmi_data = ak.macro_china_pmi()
    pmi_data.to_csv("data/macro_data/pmi.csv", index=False, encoding="utf-8-sig")
    print("PMI数据获取完成")
```

#### 4. 获取并存储 CPI 和 PPI 数据

```python
    # 获取CPI数据
    cpi_data = ak.macro_china_cpi()
    # 获取PPI数据
    ppi_data = ak.macro_china_ppi()
    
    # 合并CPI和PPI数据
    cpi_ppi_data = pd.merge(cpi_data, ppi_data, on="时间", how="outer")
    cpi_ppi_data.to_csv("data/macro_data/cpi_ppi.csv", index=False, encoding="utf-8-sig")
    print("CPI和PPI数据获取完成")
```

#### 5. 获取并存储社融数据

```python
    # 获取社会融资规模数据
    social_financing_data = ak.macro_china_shrzgm()

    print("社会融资规模数据获取完成")
```

#### 6. 获取并存储货币供应量数据


## 技术要点与最佳实践

1. **数据编码处理**：使用 `utf-8-sig` 编码存储 CSV 文件，确保中文数据正确显示
2. **目录自动创建**：使用 `os.makedirs` 并设置 `exist_ok=True`，确保目录结构自动创建
3. **执行状态反馈**：添加打印语句，实时反馈数据获取进度
4. **异常处理**：建议在生产环境中添加异常捕获机制，提高脚本稳定性
5. **数据更新策略**：可考虑添加数据更新时间戳，便于追踪数据新鲜度
