# Q-Macro 归因分析师开发方案 — 生成策略报告

## 模块简介

`report_writer.py` 是 Q-Macro 项目中的归因分析师智能体，负责生成月度投资组合报告。该报告详细解释当前的宏观状态、政策信号、市场条件以及最终的投资组合构成，帮助投资者理解策略决策过程，提升策略的透明度和可信度。

## 功能设计

### 核心功能

- **输入**：宏观状态、政策信号、市场条件、投资组合权重
- **输出**：自然语言形式的月度投资策略报告（Markdown格式）

### 报告内容结构

1. **宏观状态概览**
   - 当前经济周期（复苏、过热、滞胀、衰退）
   - 权益友好度评分及增长动量

2. **政策解读**
   - 最近支持的主要产业或主题
   - 政策支持的具体证据（引用原文关键句）

3. **市场条件分析**
   - ETF流动性状况
   - ETF拥挤度分析

4. **投资组合构建**
   - 最终ETF组合构成
   - 主题配置比例及其调整原因（如基于拥挤度调整）

  ```python
  def interpret_policy_signal(policy_signal: PolicySignal) -> str:
      if not policy_signal.use_thematic:
          return "未发现明确的主题推荐。\n"
      
      theme = policy_signal.recommended_theme
      confidence = policy_signal.confidence
      evidence = "\n".join([f"- {ev}" for ev in policy_signal.evidence])
      
      interpretation = (
          f"近期政策重点支持**{theme}**领域，具体体现在如下证据：\n"
          f"{evidence}\n"
          f"这一信号增强了我们对相关主题的信心（置信度={confidence:.2f}）。\n"
      )
      
      return interpretation
  ```

  ```python
  def analyze_market_conditions(market_cond: MarketCondition) -> str:
      liquid_etfs = ", ".join(market_cond.liquid_etfs)
  ```

  ```python
  from pydantic import BaseModel
  from typing import List, Dict

  class MacroState(BaseModel):
      regime: str
      equity_friendly_score: float
      growth_momentum: float
      inflation_momentum: float

  class PolicySignal(BaseModel):
      use_thematic: bool
      recommended_theme: str
      confidence: float
      evidence: List[str]

  class MarketCondition(BaseModel):
      liquid_etfs: List[str]
      crowded_themes: Dict[str, float]

  class Portfolio(BaseModel):
      stocks: List[Dict[str, float]]
      bonds: List[Dict[str, float]]
      commodities: List[Dict[str, float]]

  def generate_monthly_report(
      target_date: str,
      macro_state: MacroState,
      policy_signal: PolicySignal,
      market_cond: MarketCondition,
      portfolio: Portfolio
  ) -> str:
      report = (
          f"# {target_date} 投资策略报告\n\n"
          f"## 宏观状态概览\n"
          f"{describe_macro_state(macro_state)}\n"
          f"## 政策解读\n"
          f"{interpret_policy_signal(policy_signal)}\n"
          f"## 市场条件分析\n"
          f"{analyze_market_conditions(market_cond)}\n"
          f"## 投资组合构建\n"
          f"{describe_portfolio(portfolio)}\n"
      )
      
      return report
  ```
