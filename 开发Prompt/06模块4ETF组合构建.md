# ETF组合构建模块开发方案

## 模块概述

**模块4：ETF组合构建**（`src/core/portfolio_builder.py`）是 Q-Macro 策略系统的核心组件，负责基于宏观经济状态和市场校准结果，生成ETF投资组合的目标权重配置。该模块采用硬编码实现，无外部依赖，当前主题暂时固定为 "ai"。

---

## 输入参数说明

### 1. 宏观状态数据（来自 `macro_regime.py`）

```python
{
  "regime": "recovery",           # 经济周期四象限状态之一
  "equity_friendly_score": 0.68   # [0,1] 权益资产友好度评分
}
```

### 2. 市场校准结果（来自 `market_calibration.py`）

```python
{
  "liquid_etfs": ["510300", "515980", "511260", "518880"],  # 流动性合格的ETF代码列表（字符串类型）
  "crowded_adjustments": {"515980": 0.75}                   # 拥挤ETF的权重调整因子
}
```

### 3. ETF基本信息（`data/etf/etf_basic.csv`）

| 字段 | 示例 | 用途 |
|------|------|------|
| `code` | `"159790"` | ETF代码（**关键匹配字段**） |
| `csname` | `"碳中和ETF"` | 中文简称（辅助主题匹配） |
| `index_name` | `"中证内地低碳经济主题指数"` | 跟踪指数名称（**核心匹配字段**） |

**注意**：所有 `code` 字段在内存中必须统一转换为 **字符串类型**，确保匹配准确性。

## 核心实现逻辑

### 1. 大类资产基础权重确定

基于宏观经济状态（四象限模型）确定大类资产的基础配置权重：

| 经济状态 | 股票配置 | 债券配置 | 商品（黄金）配置 |
|----------|----------|----------|------------------|
| `recovery`（复苏） | 70% | 20% | 10% |
| `overheat`（过热） | 60% | 10% | 30% |
| `stagflation`（滞胀） | 30% | 30% | 40% |
| `recession`（衰退） | 40% | 50% | 10% |

**动态权重调节**：
- 股票权重根据权益友好度评分进行动态调整：
  ```python
  adjusted_stock_weight = base_stock_weight * (0.8 + 0.4 * equity_friendly_score)
  ```
- 调整后，债券和商品权重按比例缩放，确保总权重之和为100%。

### 2. 宽基ETF筛选

**宽基指数关键词列表**（硬编码）：

```python
BROAD_INDEX_KEYWORDS = [
    "沪深300", "中证全指", "上证50", "中证500", "中证800",
    "MSCI中国A股", "国证2000", "创业板指", "科创50"
]
```

- 从 `liquid_etfs` 中筛选：`index_name` 包含任一宽基指数关键词的ETF
- 若多个ETF匹配，选择第一个符合条件的ETF
- 若无匹配，回退使用默认ETF：`"510300"`（沪深300ETF）

### 3. 主题ETF筛选（当前固定为AI主题）

**AI主题关键词列表**（硬编码）：

```python
AI_THEME_KEYWORDS = [
    "人工智能", "AI", "科技", "芯片", "半导体", "数字经济",
    "云计算", "大数据", "信创", "高端制造", "机器人"
]
```

- 从 `liquid_etfs` 中筛选：
  - `index_name` **或** `csname` 包含任一AI主题关键词的ETF
- 对筛选出的主题ETF应用拥挤度调整因子：
  ```python
  adjusted_weight = base_weight * crowded_adjustments.get(code, 1.0)
  ```

### 4. 债券与商品ETF分配

| 资产类别 | 推荐ETF代码 | 要求 |
|---------|------------|------|
| 债券 | `"511260"` | 必须存在于 `liquid_etfs` 中 |
| 商品（黄金） | `"518880"` | 必须存在于 `liquid_etfs` 中 |

**处理逻辑**：
- 若指定ETF不在 `liquid_etfs` 列表中，打印警告信息并跳过该资产类别（权重归0）
- 权重分配按照步骤1确定的大类资产配置比例执行

### 5. 股票内部权重分配

- **宽基ETF**：占股票配置的60%
- **主题ETF**：占股票配置的40% × 拥挤度调整因子

### 6. 权重归一化与清理

- 移除权重 ≤ 0 的ETF
- 对剩余ETF权重进行归一化处理，确保总权重之和 = 1.0
- 保留至少2位小数精度，确保权重配置的准确性

## 模块集成

该模块作为Q-Macro策略系统的核心组件，与其他模块的集成关系如下：

- **输入**：接收来自 `macro_regime.py` 的宏观状态数据和 `market_calibration.py` 的市场校准结果
